Live site: https://namitaggarwal.github.io/

## Deployment

This repository is configured to deploy to GitHub Pages automatically from the `main` branch using GitHub Actions (`.github/workflows/deploy-pages.yml`).

To publish updates:

1. Commit changes to `main`.
2. Push to GitHub.
3. Wait for the **Deploy static site to GitHub Pages** workflow to finish.

If this is the first deployment, ensure GitHub Pages is enabled in repository settings and set to **GitHub Actions** as the source.
